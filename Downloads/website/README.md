## üö¶ Getting Started
1. **Clone the repo:**
   ```bash
   git clone https://github.com/KaarthikDassArora/phechaan.git
   cd phechaan
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run in development:**
   ```bash
   npm run dev
   ```
4. **Build for production:**
   ```bash
   npm run build
   ```
5. **Start production server (Node.js v20+):**
   ```bash
   npm start
   ```
